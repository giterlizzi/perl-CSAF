# CSAF Perl Toolkit

## Synopsis

```.pl
use CSAF;

```

## Install

Using Makefile.PL:

To install `CSAF` distribution, run the following commands.

    perl Makefile.PL
    make
    make test
    make install

Using App::cpanminus:

    cpanm CSAF
